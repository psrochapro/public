-- PROJETO: GERADOR DE CITAÇÕES PRO (JS/AI STUDIO - GITHUB PAGES)
-- DEFINIÇÃO DE ELEMENTOS E FUNCIONALIDADES

-- Principais elementos:
- Inserção de foto (Upload de arquivo ou link de imagem)
- Foto deve ter uma moldura bonita e aplicação de filtros (Grayscale, Sepia, Blur)
- Foto deve ser dimensionada automaticamente para melhor se encaixar no conteúdo
- Controle de ponto de foco (Object-fit) para evitar cortes em diferentes proporções
- Escolha de tema de cor através de um color picker (Detalhes e Fundo)
- Escolha de layout: 1:1 (Quadrado), 16:9 (Wide), 9:16 (Stories)
- Adaptação dinâmica do posicionamento da imagem e dos elementos
- Conteúdo: Imagem, Título, Citação, Autor, Ano e Logo/Marca d'água opcional
- Ajuste dinâmico de tamanho da letra e posicionamento harmônico dos elementos
- Tipografia ajustável: Estilos Serif, Sans e Script via Google Fonts
- Ajuste dinâmico das cores de texto (Contraste YIQ) de acordo com escolha de cor de tema e Fundo
- Controle de margem de segurança (Padding) para respiro visual
- Renderização via Canvas API com suporte a High-DPI (Retina displays)
- Exportação em PNG com limite de segurança de 2000px para estabilidade no navegador

-- Dinâmica de interação do usuário:
- Usuário acessa interface com painel lateral de inputs (Conteúdo, Estilo, Layout)
- Interface responsiva adaptada para hospedagem estática (GitHub Pages)
- Solicitados inputs ao usuário com histórico de cores recentes no Picker
- Preview dinâmico com sistema de zoom/escala para caber na tela do navegador
- Usuário escolhe tamanho e proporção do layout (Presets de redes sociais)
- Após validação da visualização, usuário pode salvar o PNG localmente
- Tratamento técnico de CORS para imagens externas e separação de lógica de renderização

-- Implemntações futuras (Não vão seri feitas agora - Apenas para conhecimento)
- Assistência por IA: Botão 'Surpreenda-me' (API Gemini) para sugerir citações e temas
